⚙️ 设置

🚫 网站排除列表 (黑名单)

在以下网站中，插件将完全禁用拖拽功能，以避免与网站自带的拖拽功能冲突。
每行输入一个网址或匹配模式，支持通配符 * 。

📝 匹配模式示例：

  • *.trello.com/* - 匹配所有 Trello 页面
  • github.com - 匹配 GitHub 主域名
  • *.google.com/docs/* - 匹配 Google Docs
  • example.com/admin/* - 匹配特定路径
